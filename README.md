<h2>Techstack:</h2>
<ul>
  <li><strong>Frontend:</strong> ReactJS, Vite, TypeScript, Tailwind CSS, Framer Motion</li>
  <li><strong>Backend:</strong> Supabase (PostgreSQL), Vercel Serverless Functions</li>
  <li><strong>Storage:</strong> Supabase Storage</li>
  <li><strong>Deployment:</strong> Vercel</li>
</ul>

<h2>.env File Contents:</h2>
<ul>
  <li><code>VITE_SUPABASE_URL</code> : URL of your Supabase Project</li>
  <li><code>VITE_SUPABASE_ANON_KEY</code> : Public API Key for Supabase</li>
  <li><code>SUPABASE_SERVICE_ROLE_KEY</code> : Secret Service Role Key (Required for Vercel Serverless Functions)</li>
</ul>
